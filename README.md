# RAMPAGE: Reversed Tower Defense

> Innowacyjne podejście do klasycznego gatunku Tower Defense. Zamiast budować wieże, przejmujesz kontrolę nad armią atakującą! 

Projekt w silniku Unity odwraca klasyczne założenia Tower Defense, oddając graczowi kontrolę nad armią atakującą zamiast obrońców. Celem rozgrywki jest stworzenie skutecznej kompozycji pojazdów, przełamanie linii wież przeciwnika i dotarcie do punktu końcowego mapy.
